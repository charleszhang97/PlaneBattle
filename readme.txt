  按键及鼠标功能说明：

Esc	退出
F	全屏

标题界面：
数字123 	选择难度（敌机5/30/100）
鼠标左键	选择难度

游戏界面：
Ctrl    	暂停游戏
鼠标右键 	暂停游戏
方向键 		控制飞机
WSAD    	控制飞机
移动鼠标	移动飞机
空格键  	发射子弹
鼠标左键 	发射子弹

暂停界面：
Ctrl    	取消暂停
鼠标右键 	取消暂停

结束界面：
Enter 		返回标题
Ctrl+S  	保存分数（Top10显示到排行榜）
鼠标中键	保存分数（Top10显示到排行榜）

游戏中作弊：
数字键盘-	减少一架敌机
数字键盘+	增加一架敌机
数字键盘*	发射一次全屏子弹
